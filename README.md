微信公众号、企业号Java SDK。

详细文档请看 [wiki](https://github.com/wallellen/wechat/wiki)。

## Quick Start

如果要开发公众号（订阅号、服务号）应用，在你的maven项目中添加：

```xml
<dependency>
  <groupId>com.wallellen</groupId>
  <artifactId>wechat-mp</artifactId>
  <version>1.0</version>
</dependency>
```

如果要开发企业号应用，在你的maven项目中添加：

```xml
<dependency>
  <groupId>com.wallellen</groupId>
  <artifactId>wechat-cp</artifactId>
  <version>1.0</version>
</dependency>
```
